# Phoebe mindSpace 🐶

一个温馨可爱的AI聊天应用,像忠诚的小狗一样陪伴你。

## 特性

- 🐾 可爱的腊肠狗主题设计
- 💬 实时流式对话
- 👤 用户注册与管理
- 🎨 温馨的动漫风格界面

## 运行项目

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 在浏览器打开 http://localhost:3000

## 后端配置

确保后端服务运行在 `http://localhost:8080`

如需修改API地址,请编辑 `src/config/api.js`

## 构建生产版本

```bash
npm run build
```

## 技术栈

- React 18
- Vite
- Lucide React (图标库)
